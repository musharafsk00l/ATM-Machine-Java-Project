# ATM Machine Java Project

## Project Description
This is a console-based ATM Machine application developed using Core Java.
The project simulates basic ATM functionalities such as login authentication,
balance enquiry, deposit, and withdrawal.

## Technologies Used
- Java
- Eclipse IDE
- Git & GitHub

## Features
- User authentication using Customer Number and PIN
- Checking and Saving account support
- Deposit and Withdraw functionality
- Balance enquiry
- Menu-driven console interface

## Project Structure
ATM.java        → Application entry point  
OptionMenu.java → Login and menu handling  
Account.java    → Account operations and business logic  

## How to Run
1. Open the project in Eclipse
2. Run ATM.java
3. Enter customer number and PIN
